## How to run locally
1. Install dependencies:
   `npm install`
2. Add your OpenAI key in `.env`:
   `OPENAI_API_KEY=sk-xxxxxx`
3. Start server:
   `node server/index.js`
4. Open `ai-detective/index.html` in browser and chat!

Theme: Dark + Gold consistent with your site.
